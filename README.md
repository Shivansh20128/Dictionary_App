# Dictionary App

This is a Dictionary application where a user will enter a word and the meaning of the word and the related information will be showcased. The application involves talking to a server that provides the related information.

The app uses fragments and recycler view.

The app also gives an audio option to know the correct pronunciation of the word.

# Running the app 
1. Clone the repository.

        git clone https://github.com/Shivansh20128/Dictionary_App.git
2. Open the application in Android Studio.
3. Configure the jdks as in the configuration files.
4. Run the app.
